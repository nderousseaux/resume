# Mon CV

Il s'agit de mon CV en ligne. Réalisé avec express, json-resume et [ma 
version francisé du theme stackoverflow](https://github.com/nderousseaux/jsonresume-theme-stackoverflow-french).

## Installation

```bash
npm install
```

## Utilisation

```bash
npm run dev
# ou, avec vercel
vercel dev
```

## Déploiement

```bash
vercel --prod
```
